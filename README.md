# Cervical Cancer Prediction and Classification Using Deep Learning

## Overview

This project presents an AI-powered system for Cervical Cancer Prediction and Classification using Deep Learning and Medical Image Analysis. The objective of this project is to assist in the early detection and classification of cervical cancer cells through automated image-based analysis.

## Project Objectives

* Early detection of cervical cancer using AI techniques.
* Classification of cervical cancer cell images into multiple categories.
* Improve diagnostic accuracy through Deep Learning.
* Support healthcare professionals with intelligent decision-making.

## Features

* Image Preprocessing
* Data Augmentation
* Multi-Class Classification
* Deep Learning-Based Prediction
* Medical Image Analysis
* Performance Evaluation and Visualization

## Technologies Used

* Python
* TensorFlow
* Keras
* Deep Learning
* FastAPI
* NumPy
* Pandas
* Matplotlib

## Dataset

The model was trained and evaluated using the Herlev Cervical Cancer Dataset, which contains microscopic images of cervical cells belonging to multiple  diagnostic categories. The dataset was used for image preprocessing, augmentation, training, validation, and performance evaluation of the deep learning model.
